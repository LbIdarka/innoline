Стокмастер24

Cистема работы с поставщиками wwww.sm24.innoline.pro

Система на ASP.NET для работы с поставщиками. Организация работы по скупке и распределению стока всех категорий от продуктовой линейки до строительных материалов. Имеется личный кабинет, доска объявлений, учет и формирование сделок на основе заявленных требований. Внутренняя CRM-система ведет учет поставщиков и покупателей, отслеживается активность менеджеров, выставляются необходимые производственные задачи, присутствует расширенная коммуникация между пользователями и администрацией сайта.

Используемые технологии:ASP.NET MVC, Entity Framework, Microsoft SQL, CMS Orchard, Java Script, jQuery, Bootstrap, CSS3, HTML5
